O sistema facilita o controle e organização das Escolas Bíblicas Dominicais, permitindo o cadastro digital de igrejas, professores, alunos, turmas e aulas. 
O backend foi desenvolvido em java 17 e o frontend em ReactJS com node, utilizando o vite que permite uma versão inicial mais limpa.
Fiz uso de ferramentas como o Axios: biblioteca para requisições HTTP, garantindo comunicação rápida e simplificada com APIs externas.
Tailwind CSS: framework utilitário de CSS que permite criar interfaces modernas e responsivas com produtividade e consistência.
Vite: bundler e servidor de desenvolvimento ultrarrápido, que proporciona uma experiência leve e otimizada tanto no desenvolvimento quanto na build final.
